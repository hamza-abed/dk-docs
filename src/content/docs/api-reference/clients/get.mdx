## GET /clients/`{id}`

Retourne les informations complètes d'un client, incluant ses statistiques de missions.

### Path params

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `id` | string | Oui | Identifiant unique du client. |

### Exemple de requête

```bash
curl https://api.dispatchkit.io/api/v1/clients/cli_01J5XKTY4H3GNQV2 \
  -H "Authorization: Bearer {token}"
```
