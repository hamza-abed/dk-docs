## GET /missions/`{id}`

Retourne les détails complets d'une mission, incluant l'opérateur assigné, le client, les horodatages et le reçu de complétion si disponible.

### Path params

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `id` | string (UUID) | Oui | Identifiant unique de la mission. |

### Exemple de requête

```bash
curl https://api.dispatchkit.io/api/v1/missions/c3d4e5f6-a7b8-9012-cdef-123456789012 \
  -H "Authorization: Bearer {token}"
```
