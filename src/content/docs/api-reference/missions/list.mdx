## GET /missions

Retourne la liste des missions de l'organisation active. Supporte le filtrage par date et types de tâche.

### Query params

| Paramètre | Type | Description |
|-----------|------|-------------|
| `date` | string | Filtre par date au format `YYYY-MM-DD`. Retourne les missions planifiées à cette date. |
| `task_type_ids` | string | Liste d'identifiants de types de tâche séparés par des virgules. |

<Aside type="note">
  L'API retourne un tableau JSON directement (pas de pagination ni d'enveloppe). Si aucun filtre n'est fourni, toutes les missions de l'organisation sont retournées.
</Aside>

### Exemple de requête

```bash
# Missions du 15 janvier 2024
curl "https://api.dispatchkit.io/api/v1/missions?date=2024-01-15" \
  -H "Authorization: Bearer {token}"
```
