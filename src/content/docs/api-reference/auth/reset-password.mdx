## POST /auth/reset-password

Réinitialise le mot de passe en utilisant le token reçu via l'email de réinitialisation (endpoint `POST /auth/forgot-password`).

### Body

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `token` | string | Oui | Token de réinitialisation reçu par email. |
| `new_password` | string | Oui | Nouveau mot de passe. Minimum 8 caractères. |

### Exemple de requête

```bash
curl -X POST https://api.dispatchkit.io/api/v1/auth/reset-password \
  -H "Content-Type: application/json" \
  -d '{
    "token": "abc123xyz...",
    "new_password": "NouveauMotDePasse2024!"
  }'
```

### Réponse `200 OK`

```json
{
  "message": "Mot de passe réinitialisé avec succès"
}
```

### Erreurs

| Code HTTP | Description |
|-----------|-------------|
| `400` | Token invalide, token expiré, ou mot de passe trop court |
| `500` | Erreur interne |
