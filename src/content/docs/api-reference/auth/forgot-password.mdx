## POST /auth/forgot-password

Déclenche l'envoi d'un email contenant un lien de réinitialisation de mot de passe.

<Aside type="caution">
  Pour des raisons de sécurité, cet endpoint retourne toujours `200 OK` même si l'email n'existe pas dans la base de données (protection contre l'énumération d'emails).
</Aside>

### Body

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `email` | string | Oui | Adresse email du compte à réinitialiser. |

### Exemple de requête

```bash
curl -X POST https://api.dispatchkit.io/api/v1/auth/forgot-password \
  -H "Content-Type: application/json" \
  -d '{
    "email": "marie@transport-dupont.fr"
  }'
```

### Réponse `200 OK`

```json
{
  "message": "Si cet email existe, un lien de réinitialisation a été envoyé"
}
```

### Erreurs

| Code HTTP | Description |
|-----------|-------------|
| `400` | Email manquant ou invalide |
