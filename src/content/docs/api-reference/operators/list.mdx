## GET /operators

Retourne la liste de tous les opérateurs de l'organisation active avec leur statut et leur dernière position connue.

### Query params

| Paramètre | Type | Description |
|-----------|------|-------------|
| `status` | string | Filtre par statut : `available`, `on_mission`, `unavailable`. |
| `page` | integer | Numéro de page (défaut : `1`). |
| `per_page` | integer | Résultats par page (défaut : `20`, max : `100`). |

### Exemple de requête

```bash
# Tous les opérateurs disponibles
curl "https://api.dispatchkit.io/api/v1/operators?status=available" \
  -H "Authorization: Bearer {token}"
```
