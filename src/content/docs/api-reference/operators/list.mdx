## GET /operators

Retourne la liste de tous les opérateurs de l'organisation active.

### Query params

| Paramètre | Type | Description |
|-----------|------|-------------|
| `covers_point` | string | Filtre les opérateurs dont la zone de couverture couvre un point géographique. Format : `lng,lat` (ex: `2.3522,48.8566`). |

<Aside type="note">
  L'API retourne un tableau JSON directement (pas de pagination ni d'enveloppe). Tous les opérateurs de l'organisation sont retournés.
</Aside>

### Exemple de requête

```bash
# Tous les opérateurs
curl "https://api.dispatchkit.io/api/v1/operators" \
  -H "Authorization: Bearer {token}"

# Opérateurs couvrant un point précis
curl "https://api.dispatchkit.io/api/v1/operators?covers_point=2.3522,48.8566" \
  -H "Authorization: Bearer {token}"
```
