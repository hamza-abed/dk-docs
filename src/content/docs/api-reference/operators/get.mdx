## GET /operators/`{id}`

Retourne le profil complet d'un opérateur de l'organisation, incluant son statut actuel, sa dernière position et ses statistiques.

### Path params

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `id` | string | Oui | Identifiant unique de l'opérateur. |

### Exemple de requête

```bash
curl https://api.dispatchkit.io/api/v1/operators/f1e2d3c4-b5a6-7890-abcd-ef1234567890 \
  -H "Authorization: Bearer {token}"
```
