## POST /clients

Crée un nouveau client dans l'organisation active.

### Body

<ParamField body="name" type="string" required>
  Nom du client (raison sociale ou nom complet). Max 200 caractères.
</ParamField>

<ParamField body="email" type="string" required>
  Adresse email du client. Utilisée pour les notifications et l'accès au portail client.
</ParamField>

<ParamField body="phone" type="string">
  Numéro de téléphone au format international.
</ParamField>

<ParamField body="address" type="string">
  Adresse principale du client (adresse postale complète).
</ParamField>

<ParamField body="reference" type="string">
  Code de référence interne pour synchronisation avec votre CRM ou système de facturation.
</ParamField>

### Exemple de requête

```bash
curl -X POST https://api.dispatchkit.io/api/v1/clients \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Entreprise Martin SA",
    "email": "contact@martin-sa.fr",
    "phone": "+33145678900",
    "address": "15 Avenue des Champs-Élysées, 75008 Paris",
    "reference": "CLIENT-0042"
  }'
```
