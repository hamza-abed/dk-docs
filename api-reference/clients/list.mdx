## GET /clients

Retourne la liste paginée des clients de l'organisation active.

### Query params

<ParamField query="search" type="string">
  Recherche textuelle sur le nom, l'email ou la référence du client.
</ParamField>

<ParamField query="page" type="integer">
  Numéro de page (défaut : `1`).
</ParamField>

<ParamField query="per_page" type="integer">
  Résultats par page (défaut : `20`, max : `100`).
</ParamField>

### Exemple de requête

```bash
curl "https://api.dispatchkit.io/api/v1/clients?search=Martin" \
  -H "Authorization: Bearer {token}"
```
