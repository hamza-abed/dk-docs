## GET /operators/{id}

Retourne le profil complet d'un opérateur de l'organisation, incluant son statut actuel, sa dernière position et ses statistiques.

### Path params

<ParamField path="id" type="string" required>
  Identifiant unique de l'opérateur.
</ParamField>

### Exemple de requête

```bash
curl https://api.dispatchkit.io/api/v1/operators/usr_01J5XKTY4H3GNQV2 \
  -H "Authorization: Bearer {token}"
```
