---
title: Mettre à jour un opérateur
description: Modifie les informations d'un opérateur.
---

## PUT /operators/{id}

Met à jour les informations d'un opérateur. Seuls les managers, admins et owners peuvent modifier les opérateurs.

### Path params

<ParamField path="id" type="string" required>
  Identifiant unique de l'opérateur.
</ParamField>

### Body

<ParamField body="first_name" type="string">
  Prénom de l'opérateur.
</ParamField>

<ParamField body="last_name" type="string">
  Nom de famille de l'opérateur.
</ParamField>

<ParamField body="phone" type="string">
  Numéro de téléphone au format international.
</ParamField>

<ParamField body="vehicle_plate" type="string">
  Numéro d'immatriculation du véhicule.
</ParamField>

### Exemple de requête

```bash
curl -X PUT https://api.dispatchkit.io/api/v1/operators/usr_01J5XKTY4H3GNQV2 \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{
    "phone": "+33698765432",
    "vehicle_plate": "ZY-987-AB"
  }'
```

### Réponse

```json
{
  "id": "usr_01J5XKTY4H3GNQV2",
  "first_name": "Jean",
  "last_name": "Dupont",
  "email": "jean.dupont@transport-dupont.fr",
  "phone": "+33698765432",
  "vehicle_plate": "ZY-987-AB",
  "updated_at": "2024-01-15T11:00:00Z"
}
```
