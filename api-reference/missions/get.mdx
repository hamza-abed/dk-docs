## GET /missions/{id}

Retourne les détails complets d'une mission, incluant l'opérateur assigné, le client, les horodatages et le reçu de complétion si disponible.

### Path params

<ParamField path="id" type="string" required>
  Identifiant unique de la mission (ex. : `mis_01J5XKTY4H3GNQV2`).
</ParamField>

### Exemple de requête

```bash
curl https://api.dispatchkit.io/api/v1/missions/mis_01J5XKTY4H3GNQV2 \
  -H "Authorization: Bearer {token}"
```
